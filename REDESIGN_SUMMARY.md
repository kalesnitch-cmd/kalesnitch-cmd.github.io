## 🎨 Ключевые изменения

### Цветовая палитра
```css
/* Основные цвета */
--card-bg: rgba(255, 255, 255, 0.02)
--card-border: rgba(255, 255, 255, 0.055)
--text-primary: #f9fafb
--text-secondary: #4b5563
--text-tertiary: #374151

/* Акцентные цвета */
--accent-yellow: #fbbf24  /* Калории */
--accent-blue: #60a5fa    /* Белок */
--accent-orange: #fb923c  /* Жиры */
--accent-green: #4ade80   /* Углеводы */
```

### Типографика
- **Заголовки**: Unbounded (800-900 weight)
- **Основной текст**: Manrope (500-700 weight)
- **Размеры**: 9px - 28px (уменьшены на 20-30%)
- **Letter-spacing**: -0.01em для улучшенной читаемости

### Радиусы скругления
```css
--radius-xs: 5px    /* Чипы */
--radius-sm: 7px    /* Маленькие элементы */
--radius-md: 9px    /* Инпуты */
--radius-lg: 12px   /* Кнопки */
--radius-xl: 17px   /* Карточки */
--radius-xxl: 20px  /* Большие карточки */
--radius-xxxl: 26px /* Модальные окна */
```

### Компоненты

#### Кнопки
- Уменьшены размеры: padding 12px 20px (было 16px 28px)
- Шрифт: 13px (было 16px)
- Тени: var(--shadow-sm) и var(--shadow-md)
- Переходы: 0.15s

#### Карточки
- Фон: rgba(255, 255, 255, 0.02)
- Граница: 1px solid rgba(255, 255, 255, 0.055)
- Радиус: 17px
- Padding: 18px (было 24px)

#### Модальные окна
- Фон: linear-gradient(160deg, #0b1520, #060e18)
- Радиус: 26px
- Backdrop-filter: blur(16px)
- Тень: 0 48px 120px rgba(0, 0, 0, 0.95)

## 🚀 Новые возможности

### 1. Чипы и бейджи
```html

  ОСНОВНАЯ ТРЕНИРОВКА

```

### 2. Иконки-контейнеры
```html
<div class="icon-container">🏋️</div>
<div class="icon-container-lg">💪</div>
```

```html
<div class="meal-card">

    <div class="icon-container">🏋️</div>

  </div>

    <div class="meal-card-stat">Вес 80кг</div>
    <div class="meal-card-stat">Отдых 2мин</div>
  </div>

```

# 💪 Тренировочный Дневник - Telegram Mini App + PWA

Полноценное приложение для отслеживания силовых тренировок в Telegram с AI тренером и полной кастомизацией планов. Также работает как отдельное PWA приложение на iOS/Android.

## ✨ Основные возможности

### 🏋️ Тренировки
- **2 готовые программы:** Full Body и Ягодицы
- **3 дня тренировок** в каждой программе (А, Б, В)
- **Запись весов и повторений** для каждого подхода
- **Автосохранение** - данные не теряются при выходе
- **Добавление подходов** кнопкой "+" прямо во время тренировки
- **Техника выполнения** с видео для каждого упражнения
- **Группы мышц** указаны для каждого упражнения

### 🤖 AI Тренер
- **Персональные планы** на основе твоих данных
- Учитывает: пол, возраст, вес, рост, опыт, цель, оборудование
- **Структурированный формат** - AI планы работают как обычные программы
- **Быстрая генерация** через Groq API (Llama 3.3 70B)

### 📝 Мой план
- **Полное редактирование** программ тренировок
- **Добавление упражнений** из справочника (70+ упражнений)
- **Создание своих упражнений** с любым названием
- **Изменение порядка** упражнений (⬆️ ⬇️)
- **Редактирование** названий, подходов, повторений
- **Сброс к стандартному** плану

### 📅 История
- **Все тренировки** с датами
- **Детальная информация** по каждому упражнению
- **Редактирование** прошлых тренировок
- **Объединение** тренировок за один день

### 📈 Прогресс
- **Выбор программы** (Full Body / Ягодицы)
- **Максимальные веса** для каждого упражнения
- **Группировка по дням** (А, Б, В)
- **Счётчики:** всего тренировок, дней подряд

### 📚 Гайды
- **70+ упражнений** с описанием
- **Видео техники** для каждого упражнения
- **Группы мышц** для каждого упражнения
- **Категории:** Грудь, Спина, Плечи, Трицепс, Бицепс, Предплечья, Ноги, Икры, Пресс

### 🎨 Дизайн
- **Спокойная тёмная тема** (#0f0f0f)
- **Градиентные кнопки** с фиолетовым акцентом
- **Современный минимализм**
- **Адаптивный дизайн** для мобильных
- **Кнопка "Готово"** для закрытия клавиатуры на iOS

### 📱 PWA - Progressive Web App
- **Установка на домашний экран** iOS/Android
- **Работает офлайн** с кэшированием
- **Нативный вид** как обычное приложение
- **Совместимость** с Telegram Mini App
- **Быстрая загрузка** благодаря Service Worker

## 🚀 Деплой

### Vercel (рекомендуется)

1. **Установи Vercel CLI:**
   ```bash
   npm i -g vercel
   ```

2. **Задеплой проект:**
   ```bash
   cd training_bot
   vercel --prod
   ```

3. **Добавь переменные окружения:**
   - `GROQ_API_KEY` - API ключ от https://console.groq.com

4. **Настрой бота через @BotFather:**
   ```
   /newapp
   ```
   - Введи URL: `https://твой-проект.vercel.app`

5. **Создай иконки для PWA:**
   - Открой `create-icons.html` в браузере
   - Скачаются `icon-192.png` и `icon-512.png`
   - Загрузи их в корень проекта на Vercel

## 📱 Установка как PWA

### iOS (Safari)
1. Открой приложение в Safari
2. Нажми кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Выбери "На экран Домой"
4. Готово! Приложение появится на домашнем экране

### Android (Chrome)
1. Открой приложение в Chrome
2. Нажми кнопку "📱 Установить приложение"
3. Подтверди установку
4. Готово! Приложение появится в списке приложений

## 💾 Хранение данных

Все данные в `localStorage`:
- `workoutHistory` - история тренировок
- `userWorkoutProgram` - кастомный план Full Body
- `userGlutesProgram` - кастомный план Ягодицы
- `aiWorkoutPlans` - сохранённые AI планы
- `currentWorkoutInProgress` - текущая незавершённая тренировка
- `appTheme` - выбранная тема
- `progressProgram` - выбранная программа в разделе "Прогресс"

## 🎓 Программы тренировок

### 💪 Full Body
**Цель:** Развитие всего тела с акцентом на жим лёжа  
**Частота:** 3 раза в неделю

- **День А - Тяжёлый жим:** 7 упражнений
- **День Б - Объёмный жим + горизонтальная тяга:** 7 упражнений
- **День В - Средний + акцент на ноги:** 8 упражнений

### 🍑 Ягодицы
**Цель:** Развитие ягодиц и ног  
**Частота:** 3 раза в неделю

- **День А - Ягодицы + Задняя поверхность:** 5 упражнений
- **День Б - Квадрицепсы + Боковая поверхность:** 5 упражнений
- **День В - Форма + Стабилизация:** 7 упражнений

## 🐛 Troubleshooting

- **Клавиатура не закрывается на iOS:** Используй кнопку "✓ Готово" внизу экрана
- **Данные не сохраняются:** Проверь, что localStorage не заблокирован
- **AI не работает:** Проверь API ключ в переменных окружения Vercel
- **Упражнение сворачивается при клике:** Обнови до последней версии
- **PWA не устанавливается:** Убедись, что сайт открыт через HTTPS
- **Иконки не отображаются:** Загрузи `icon-192.png` и `icon-512.png` в корень проекта
